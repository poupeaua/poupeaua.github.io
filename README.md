# Professional Website

The website is meant to be simple and easy to navigate in order to
present my qualifications, projects and contact information. 

## Sections

It is divided into 3 sections:

1. Home
2. Projects
3. Contact

## Technical Details

- Built using [Vite](https://vitejs.dev/)
- Uses [React](https://reactjs.org/)
- Uses [TailwindCSS](https://tailwindcss.com/)
- For UI [Shadcn](https://ui.shadcn.com/)
